# jsz - 小红书爆款图片抓取工具

自动搜索小红书指定关键词，下载高点赞笔记的封面图片。

## 功能

- 🔍 关键词搜索小红书笔记
- 📊 按点赞数排序，取 Top N
- 🖼️ 自动下载封面图片
- 📁 按日期归档

## 安装

```bash
pip install -r requirements.txt
playwright install chromium
```

## 使用

```bash
python scraper.py
```

## 配置

编辑 `config.py`：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| KEYWORDS | 搜索关键词列表 | ["金手镯", "黄金手镯", "金镯子"] |
| TOP_N | 每个关键词取 Top N | 15 |
| HEADLESS | 无头模式 | True |
| SCROLL_TIMES | 搜索页滚动次数 | 3 |

## 输出

图片保存在 `images/` 目录下，按日期归档：
```
images/
  └── 2026-06-23/
        ├── 金手镯_01_某标题.jpg
        ├── 金手镯_02_某标题.jpg
        └── ...
```
